# Research and Development Sand-Box

---
## Local Development

Create environment variables in your shell:

```
pipenv install --dev
```

Activate:
```
pipenv shell
```

---

## Notebooks Formatting 

See [https://github.com/nbQA-dev/nbQA](nbQA)

- isort

```
nbqa isort my_notebook.ipynb
```

- black

```
nbqa black my_notebook.ipynb
```
